Inari-Kontroller の手順書

1. この ZIP を右クリックして「すべて展開」。
2. フォルダー直下の Inari_Kontroller.exe を起動。
3. HMDをPCと接続した状態で、SteamVR と VRChat を起動。
4. アバターを操作可能な状態で、入力を見る画面を開き左右スティック入力が反応することを確認してください。
5. 「回して故障診断」を開き、「計測開始」。
6. オレンジのガイドに合わせて、スティックを操作してください。

注意:
- この公開版は SteamVR ドライバーをインストール・登録しません。
- 診断と補正処理はローカルで行われ、コントローラー入力データを外部サーバーへ送信しません。
- 「入力を見る」は現在の入力確認だけに使います。診断や補正作成は「回して故障診断」で行います。
- VRChat OSC補正は、診断後に必要な場合だけ「出力設定」の「VRChatへ補正入力を送る」から開始してください。
- SteamVR自動起動を有効化すると、SteamVR起動時にVRChat OSC補正が始まります。不要な場合は有効化しないでください。
- アプリを閉じると、手動起動したエンジンとOSC送信は停止します。
- このアプリは一時的に操作を補助する目的のツールです。物理的なスティック故障の修理を目的としたものではないため、症状がある場合は早めの修理や交換をおすすめします。
- 現在確認できている環境は Quest 2 + Virtual Desktop + SteamVR + VRChat の組み合わせのみです。ほかの環境での動作報告や不具合報告を歓迎します。

フォルダー構成:
- Inari_Kontroller.exe: 起動用
- app: アプリ本体と必要DLL
- docs: ライセンス、第三者ソフトウェア表記、第三者ライセンス本文、解除・トラブル時の戻し方
