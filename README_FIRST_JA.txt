VR Stick Scope を使う前に

1. このZIPを右クリックして「すべて展開」してください。
2. フォルダ直下の 00_START_VRStickScope.exe を起動してください。
3. SteamVR と VRChat を起動し、HMDを装着してAFKではない状態にしてください。
4. VRChat上で、スティック入力で移動や旋回が反応することを確認してください。
5. 「回して故障診断」を開き、「計測開始」を押してください。
6. オレンジの目標点を追ってスティックを3周回してください。

注意:
- この公開版は SteamVR ドライバーをインストール・登録しません。
- 診断と補正処理はローカルで行われ、コントローラー入力データを外部サーバーへ送信しません。
- 「入力を見る」はライブ表示だけです。故障判定や補正作成は行いません。
- 「回して故障診断」は、計測開始時に診断エンジンを自動で起動し、完了または中止時に停止します。
- VRChat OSC補正は、診断後に必要な場合だけ Runtime の「VRChatへ補正入力を送る」から開始してください。
- アプリを閉じると、手動起動したエンジンとOSC送信は停止します。
- この公開版では SteamVR 自動起動の新規有効化はできません。古い版の自動起動が残っている場合だけ Runtime から解除できます。

フォルダ構造:
- 00_START_VRStickScope.exe: 起動用
- app: アプリ本体と必要DLL
- docs: ライセンスと第三者ソフトウェア表記
