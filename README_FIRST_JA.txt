VR Stick Scope を使う前に

1. この ZIP を右クリックして「すべて展開」してください。
2. フォルダー直下の 00_START_VRStickScope.exe を起動してください。
3. SteamVR と VRChat を起動してください。
4. VRChat が入力を受け付ける状態で、スティック入力に移動や旋回が反応することを確認してください。
5. 「回して故障診断」を開き、「計測開始」を押してください。
6. オレンジの目標点を追って、スティックを3周回してください。

注意:
- この公開版は SteamVR ドライバーをインストール・登録しません。
- 診断と補正処理はローカルで行われ、コントローラー入力データを外部サーバーへ送信しません。
- 「入力を見る」は現在の入力確認だけに使います。診断や補正作成は「回して故障診断」で行います。
- VRChat OSC補正は、診断後に必要な場合だけ「出力設定」の「VRChatへ補正入力を送る」から開始してください。
- SteamVR自動起動を有効化すると、SteamVR起動時にVRChat OSC補正が始まります。不要な場合は有効化しないでください。
- アプリを閉じると、手動起動したエンジンとOSC送信は停止します。

フォルダー構成:
- 00_START_VRStickScope.exe: 起動用
- app: アプリ本体と必要DLL
- docs: ライセンス、第三者ソフトウェア表記、戻し方
