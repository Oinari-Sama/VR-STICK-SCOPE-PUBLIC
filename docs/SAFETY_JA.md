# 安全メモ

Inari-Kontroller は実験的な診断・補正ツールです。VRChat、SteamVR、Meta、Valve の公式ツールではありません。

- この公開版は SteamVR ドライバーをインストール・登録しません。
- OpenVR 経由でコントローラー入力を読み取ります。
- 必要な場合だけ、VRChat OSC へ補正済み入力を送信します。
- OSC 送信は `出力設定` から手動で開始できます。
- SteamVR自動起動を有効化した場合は、SteamVR 起動時に VRChat OSC 補正が始まります。
- 回して故障診断のエンジンは、計測開始時に起動し、完了または中止時に停止します。
- 入力を見る画面のエンジンは、画面を開いた時に起動し、画面を離れた時に停止します。
- アプリを閉じると、手動起動したエンジンと OSC 送信は停止します。
- 診断・補正処理はローカルで行われます。
- コントローラー入力データを外部サーバーへアップロードしません。
- このアプリは一時的に操作を補助する目的のツールです。物理的なスティック故障の修理を目的としたものではないため、症状がある場合は早めの修理や交換をおすすめします。

現在確認できている環境は Quest 2 + Virtual Desktop + SteamVR + VRChat の組み合わせのみです。ほかの環境での動作報告や不具合報告を歓迎します。

入力がおかしくなった場合は、Inari-Kontroller を終了し、VRChat と SteamVR を再起動してください。SteamVR自動起動を有効化していた場合は、`出力設定` から `SteamVR自動起動を解除` を押してください。

プロファイルとローカル設定は、ユーザーのローカルアプリケーションデータ内の `InariKontroller` に保存されます。
