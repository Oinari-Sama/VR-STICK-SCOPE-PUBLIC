# 安全メモ

VR Stick Scope は実験的な診断・補正ツールです。VRChat、SteamVR、
Meta、Valve の公式ツールではありません。

- この公開版は SteamVR ドライバーをインストール・登録しません。
- OpenVR 経由でコントローラー入力を読み取ります。
- 必要な場合だけ、VRChat OSC へ補正済み入力を送信します。
- 診断・補正処理はローカルで行われます。
- コントローラー入力データを外部サーバーへアップロードしません。
- 補正は物理故障を修理するものではありません。

入力がおかしくなった場合は、VR Stick Scope を終了し、VRChat と SteamVR
を再起動してください。SteamVR 自動起動を有効にした場合は、Runtime 画面
から無効化できます。

プロファイルとローカル設定は、ユーザーのローカルアプリケーションデータ
フォルダ内の `VRStickScope` に保存されます。
