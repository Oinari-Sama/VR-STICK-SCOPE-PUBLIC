# VR Stick Scope 使い方

VR Stick Scope は、Quest コントローラーのスティック入力を確認し、VRChat OSC へ補正入力を送るためのツールです。SteamVR の追加ドライバーなしで使えます。

## まず起動する

1. ZIP を右クリックして、すべて展開します。
2. 展開したフォルダーの `00_START_VRStickScope.exe` を起動します。
3. SteamVR と VRChat を起動します。
4. VRChat が入力を受け付ける状態にします。
5. VRChat 上でスティック入力に移動や旋回が反応することを確認します。

## スティックを調べる

`回して故障診断` を開き、オレンジの目標点に合わせてスティックをゆっくり回してください。

左スティックは時計回り、右スティックは反時計回りです。ガイドから急に中心へ落ちたり、反対方向へ飛んだりする入力を検出し、方向ごとの補正データを作成できます。

`入力を見る` は現在の入力を確認するための画面です。診断や補正作成は `回して故障診断` で行います。

## VRChat OSC 補正を試す

1. VRChat 側で OSC を有効にします。
2. VRChat が入力を受け付ける状態にします。
3. VR Stick Scope の `出力設定` を開きます。
4. `VRChatへ補正入力を送る` を押します。
5. VRChat 上で移動入力と左右旋回を確認します。

送信する OSC 入力は次の3つです。

```text
/input/Horizontal      左右移動
/input/Vertical        前後移動
/input/LookHorizontal  左右旋回
```

停止するときは `VRChat出力を停止` を押します。アプリ終了時にも OSC 入力を0へ戻します。

## SteamVR 自動起動

`出力設定` の `SteamVR自動起動を有効化` を押すと、SteamVR 起動時に VRChat OSC 補正が自動で始まります。

便利ですが、不要なときにも補正入力を送る原因になります。毎回手動で開始したい場合は有効化しないでください。

解除する場合:

1. `出力設定` を開きます。
2. `SteamVR自動起動を解除` を押します。
3. VR Stick Scope、SteamVR、VRChat を終了します。
4. SteamVR と VRChat を起動し直します。

## 注意

- 補正を使うには、VRChat 側の OSC を有効にする必要があります。
- 診断時は、VRChat が入力を受け付ける状態で操作してください。
- アプリを閉じると、手動起動したエンジンと OSC 送信は停止します。
- Quest、Virtual Desktop、SteamVR の構成によって軸の見え方が違う場合があります。
- 最初は診断だけで動作を確認してください。
