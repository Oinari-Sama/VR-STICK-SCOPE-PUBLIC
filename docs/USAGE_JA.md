# Inari-Kontroller 使い方

Inari-Kontroller は、Quest コントローラーのスティック入力を確認し、VRChat OSC へ補正入力を送るためのツールです。SteamVR の追加ドライバーなしで使えます。

## まず起動する

1. ZIP を右クリックして、すべて展開します。
2. 展開したフォルダーの `Inari_Kontroller.exe` を起動します。
3. HMDをPCと接続した状態で、SteamVR と VRChat を起動します。
4. アバターを操作可能な状態にします。
5. `入力を見る` 画面を開き、左右スティック入力が反応することを確認します。

## スティックを調べる

`回して故障診断` を開き、`計測開始` を押してください。

左スティックは時計回り、右スティックは反時計回りです。オレンジのガイドに合わせてスティックを操作してください。ガイドから急に中心へ落ちたり、反対方向へ飛んだりする入力を検出し、方向ごとの補正データを作成できます。

`入力を見る` は現在の入力を確認するための画面です。診断や補正作成は `回して故障診断` で行います。

## VRChat OSC 補正を試す

1. VRChat 側で OSC を有効にします。
2. アバターを操作可能な状態にします。
3. Inari-Kontroller の `出力設定` を開きます。
4. `VRChatへ補正入力を送る` を押します。
5. VRChat 上で移動入力と左右旋回を確認します。

送信する OSC 入力は次の3つです。

```text
/input/Horizontal      左右移動
/input/Vertical        前後移動
/input/LookHorizontal  左右旋回
```

停止するときは `VRChat出力を停止` を押します。アプリ終了時にも OSC 入力を0へ戻します。

## SteamVR 自動起動

`出力設定` の `SteamVR自動起動を有効化` を押すと、SteamVR 起動時に VRChat OSC 補正が自動で始まります。

便利ですが、不要なときにも補正入力を送る原因になります。毎回手動で開始したい場合は有効化しないでください。

解除する場合:

1. `出力設定` を開きます。
2. `SteamVR自動起動を解除` を押します。
3. Inari-Kontroller、SteamVR、VRChat を終了します。
4. SteamVR と VRChat を起動し直します。

## 確認済み環境とフィードバック

現在確認できている環境は Quest 2 + Virtual Desktop + SteamVR + VRChat の組み合わせのみです。ほかの環境での動作報告や不具合報告を歓迎します。

## 注意

- 補正を使うには、VRChat 側の OSC を有効にする必要があります。
- 診断時は、アバターを操作可能な状態で操作してください。
- アプリを閉じると、手動起動したエンジンと OSC 送信は停止します。
- このアプリは一時的に操作を補助する目的のツールです。物理的なスティック故障の修理を目的としたものではないため、症状がある場合は早めの修理や交換をおすすめします。
- Quest、Virtual Desktop、SteamVR の構成によって軸の見え方が違う場合があります。
- 最初は診断だけで動作を確認してください。
